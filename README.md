# paliativa
Este é o repositório da landing page oficial da PaliAtiva, um serviço de consultoria focado na implementação e aperfeiçoamento de cuidados paliativos em serviços e instituições de saúde. O objetivo principal do site é apresentar os serviços, a metodologia de trabalho e o perfil do profissional responsável, além de captar contatos via WhatsApp.
