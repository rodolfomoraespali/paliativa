# PaliAtiva - Consultoria em Cuidados Paliativos

Este é o repositório da landing page oficial da **PaliAtiva**, um serviço de consultoria focado na implementação e aperfeiçoamento de cuidados paliativos em serviços e instituições de saúde. O objetivo principal do site é apresentar os serviços, a metodologia de trabalho e o perfil do profissional responsável, além de captar contatos via WhatsApp.

---

## 🚀 Visão Geral do Projeto

A página foi desenvolvida para ser uma vitrine profissional, responsiva e direta ao ponto. Ela substitui a antiga estrutura de "mentoria" por um foco institucional ("B2B" - Business to Business), visando hospitais, clínicas e gestores de saúde.

### 📌 Principais Seções
* **Hero (Início):** Apresentação de impacto com imagem de fundo e overlay transparente, destacando o propósito da consultoria.
* **Nossos Serviços:** Detalhamento das frentes de atuação (Capacitação, Implementação, Protocolos, Indicadores).
* **Métodos de Trabalho:** Como a consultoria é aplicada na prática (Aulas, Casos Clínicos, Treinamentos, etc.).
* **Quem Conduz:** Perfil do Dr. Rodolfo Moraes, suas credenciais médicas e vasto portfólio de experiência em estruturação de serviços de saúde.
* **Fale Conosco:** Formulário de contato dinâmico que gera uma mensagem formatada e envia direto para o WhatsApp comercial.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais (Vanilla), sem a necessidade de frameworks complexos, garantindo máxima performance e facilidade de manutenção:

* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização responsiva, CSS Variables (Custom Properties), Flexbox, CSS Grid e efeitos visuais (Gradients, Glassmorphism).
* **JavaScript (Vanilla):** Lógica para rolagem suave (smooth scroll), formatação do formulário de contato e sistema de notificações (Toast).
